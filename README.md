# APSMnet
**Long-term series wetland classification of Guangdong-Hong Kong-Macao Greater Bay Area based on APSMnet**<br />
Run “train_2000.py” to train the model.<br /> 
Run “classify_2000.py” to classify wetlands in GBA.<br />
